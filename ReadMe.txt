1- Clone the repository:
git clone https://github.com/elmehdifaida/University-Booking-System.git

2- Navigate to the project directory:
cd University-Booking-System

3- Install dependencies:
npm Install

4- Check for environment variables:
create .env file
put requirement variables such as : 
SecretToken = 'TokenSecret'
EMAILPASS = app access password of your gmail account
YOUREMAIL = you email 

-5 Run the project
npm start
